# Infra for Vanacular
Backend infra is built on [Nest](https://github.com/nestjs/nest) and typescript. This entails all the services and endpoints for the vanacular language learning app.


## Project setup
Use Docker or Nest runtime:

```bash
# do first
$ npm install
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deployment

- Vanacular is currently deployed on [Render](https://render.com) for free with Docker.
- Neon is used as Postgres Db.
- Upstash for Redis.
